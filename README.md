# Siddhant Nagraj — Portfolio

Personal portfolio of Siddhant Nagraj, Product Designer.

A static site — no build step, no dependencies. Just open `index.html`.

## Structure
- `index.html` — entry point
- `*.jsx` / `*.css` — app code and styles (React via in-browser Babel)
- `assets/` — images, case-study screens, résumé, social card

## Deploy
Static hosting. On Vercel, deploy as a static project (Framework Preset: **Other**) — `index.html` is served automatically.
